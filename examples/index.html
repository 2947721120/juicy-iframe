<juicy-iframe>

<juicy-iframe> (precisely: <template is="juicy-iframe">) is a custom element that lets you stamp inline or external HTML into an iframe. It also provides simple data binding, that works for native JS/HTML as well as for Polymer's dom-bind. See README.md for more information.
